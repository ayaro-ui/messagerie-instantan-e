# Messagerie Instantanée

Application de messagerie instantanée en temps réel avec React et Node.js.

## Stack Technique
- **Frontend** : React.js
- **Backend** : Node.js + Express + Socket.io
- **Base de données** : MySQL (Sequelize)
- **Auth** : JWT + bcrypt

## Installation

### Backend
```bash
cd server
npm install
node server.js
```

### Frontend
```bash
cd client
npm install
npm start
```

## Fonctionnalités
- Authentification (inscription / connexion)
- Messagerie en temps réel avec Socket.io
- Liste des utilisateurs connectés
